Alumn@: Martin Ingaramo

  1. Que es un sistema de control de versiones?

    Son herramientas que ayudan a equipos a trabajar de forma colaborativa manejando los cambios de codigos. Permitiendo que se puedan seguir todos estos cambios simultaneos de forma ordenada y sin riesgos de perder trabajo realizado.

  2. Cual es la diferencia entre git y github?

    Git es una herramienta de versionado para controlar los cambios sobre los archivos, mientras que GitHub es un sistema que permite alojar estos archivos.

  3. Cómo creo una rama en git?

    Primero me ubico en el Master, traigo la ultima version disponible y creo la rama con el comando "git checkout -b nombreDeLaRama" ej: git checkout -b add/branchMartinIngaramo
